IAG ToolKIT Welcome!

The aim of this application is to provide a quick, effortless view of the current state of the Management Action Plan validation process across all of IAG.

Do Not Close the Terminal Window: The use of this toolkit requires a light-weight web server to run which you can see in the terminal window that was launched when you ran the toolkit. If you close that terminal, you are effectively exiting the application and the toolkit will no longer function. Attempting to use the toolkit after you closed the terminal will result in a 'Page Not Available' or similar message. Only close the terminal when you are finished using the toolkit.
Basic Usage

  • Check Your Access

    Make sure you have access to these locations:

    {{ FINDING_TRACK_REPORT }}
    {{ OTHER_REPORT }}

  • Sync Your Database

    This toolkit is dependent on a Finding Track report which a MAP Champion will replace at the following directory on a weekly basis

    {{ FINDING_TRACK_REPORT }}

    Keep an eye on the Last Data Refresh: {{ DB_REFRESH_DATE }} label, found in the left-hand column, to ensure your database is current. If you need to update the database, do so by clicking on Sync Database found under the dropdown menu in the top right corner. This date reflects the creation date of the Finding Track report. So if the report never changes this date will never change. Be sure the MAP Champion is replacing the report.

  • Configure Your Settings

    Many of the reports in this toolkit depend on an initial configuration from the user. Be sure you go to Team Settings, also found in the dropdown menu in the top right corner, and specify who you are, who is on your audit team, and who your team lead and peers are. Doing so will enable more accurate reporting as Finding Track allows for multiple reviewers on a MAP. These settings will allow the reports to be filtered down to only those team members you want to track.
